# DZ_TP

This repository contains practical tasks for the "TP" course.

## Project structure

- `1/`
  - `1.py` - pedometer calculator (speed, steps, and simple health message).
  - `2.py` - renovation calculator (wallpaper/paint and total cost).
- `2/`
  - `practical_work_2.py` - console app for products, filtering, meal logging, and nutrition totals.
  - `meals_history.json` - saved meal history for the console app.

## Run

Requirements:
- Python 3.10+

Examples:

```bash
python 1/1.py
python 1/2.py
```

For practical work 2, run from folder `2/` so history is read/written to `2/meals_history.json`:

```bash
cd 2
python practical_work_2.py
```
